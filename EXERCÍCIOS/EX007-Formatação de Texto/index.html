Principais formatações

Negrito/Destaque

Nesta frase temos um termo em negrito usando a tag <b>

Nesta frase temos um termo em destaque usando a tag <strong>

Nesta frase temos um termo em italico usando a tag <i>

Nesta frase temos um termo enfatizado usando a tag <em>

Texto marcado

Podemos criar também um texto marcado usando a tag <mark>

E no outro parágrafo temos outro texto marcado no final

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno neste parágrafo

Texto excluído

Com esta tag podemos indicar um texto excluído, para mostrar que ele pode ser lido, mas será desconsiderado. Para isso usamos a tag <del>

Texto inserido

Neste parágrafo mostramos um texto inserido que deve ser lido, e dever ser considerado. Para isso usamos a tag <ins>

Texto sobrescrito

Para inserir coisa do tipo x20=3. Utilizamos a tag <sup>

Texto subscrito

Para inserir coisas do tipo H2O. Utilizamos a tag <sup>
